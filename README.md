# Prometheus

Prometheus Drive is an open-source, self-hosted cloud storage solution inspired by platforms like Google Drive. Built on Node.js, Prometheus Drive aims to offers users the ability to store, manage, and share files securely while providing complete control over data privacy. NOTE # this project is still under active development.
